## Overview

Moving Icons includes **530+ animated icons** based on the Lucide icon set. All icons are available as Svelte components with consistent naming and behavior.

<Card title="Interactive Gallery" icon="images" href="https://movingicons.dev/icons">
  Browse and preview all icons with live animations at movingicons.dev/icons
</Card>

## Naming Convention

All icons follow PascalCase naming:

<CodeGroup>
```svelte Component Import
import { Accessibility, Activity, Airplay } from 'moving-icons';
```

```svelte File Name Pattern
accessibility.svelte → Accessibility
alarm-clock-check.svelte → AlarmClockCheck
arrow-down-0-1.svelte → ArrowDown01
```
</CodeGroup>

<Note>
  Icon names are derived from Lucide's naming conventions, with hyphens converted to PascalCase.
</Note>

## Usage

Import any icon by its PascalCase name:

<CodeGroup>
```svelte Single Import
import { Heart } from 'moving-icons';

<Heart />
```

```svelte Multiple Imports
import { Heart, Star, ThumbsUp } from 'moving-icons';

<Heart />
<Star />
<ThumbsUp />
```

```svelte With Props
import { Accessibility } from 'moving-icons';

<Accessibility
  color="#3b82f6"
  size={32}
  strokeWidth={2}
/>
```
</CodeGroup>

<Check>
  All 530+ icons share the same props and behavior pattern.
</Check>

## Lucide Consistency

Moving Icons maintains naming consistency with Lucide:

- Same icon names (kebab-case → PascalCase)
- Same visual design language
- Same 24x24 viewBox grid
- Enhanced with custom animations

<Tip>
  If you're familiar with Lucide icons, you already know the Moving Icons naming convention!
</Tip>
