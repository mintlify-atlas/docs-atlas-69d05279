## Why Moving Icons?

Moving Icons brings delightful, purposeful animation to your Svelte applications without the complexity. Each icon is carefully crafted to enhance user experience through subtle, meaningful motion.

### Key Features

<AccordionGroup>
  <Accordion title="Zero Dependencies" icon="package">
    Built purely with Svelte 5 and vanilla CSS animations. No external animation libraries required, keeping your bundle size minimal.
  </Accordion>
  
  <Accordion title="Fully Tree-Shakeable" icon="tree">
    Import only the icons you need. Dead code elimination ensures unused icons never make it to production, optimizing your bundle size.
  </Accordion>
  
  <Accordion title="Completely Customizable" icon="palette">
    Control size, color, stroke width, and animation state with simple props. Style icons to match your design system perfectly.
  </Accordion>
  
  <Accordion title="Vanilla CSS Animations" icon="sparkles">
    All animations use pure CSS with no JavaScript runtime overhead. Smooth 60fps animations that respect user preferences like reduced motion.
  </Accordion>
  
  <Accordion title="Svelte 5 Native" icon="code">
    Built from the ground up for Svelte 5 using modern features like `$state` and `$props`. Fully typed with TypeScript support.
  </Accordion>
  
  <Accordion title="MIT Licensed" icon="scale">
    Free to use in personal and commercial projects. Use them however you want, wherever you want.
  </Accordion>
</AccordionGroup>

## What Makes These Icons Special

Moving Icons isn't just another icon library. Every icon includes thoughtfully designed animations that activate on hover or on demand:

- **Purposeful Motion** - Animations enhance meaning rather than distract. A bell icon rings, a heart beats, settings rotate.
- **Performance First** - Pure CSS animations mean no JavaScript overhead and silky smooth 60fps motion.
- **Accessibility Aware** - Respects `prefers-reduced-motion` for users who need calmer interfaces.
- **Developer Friendly** - Simple, predictable API with full TypeScript support and excellent tree-shaking.

<Note>
Moving Icons are based on [Lucide icons](https://lucide.dev), ensuring you get the clean, consistent design language you already know and love, now with beautiful animations.
</Note>

## Quick Example

Get started in seconds with this simple example:

```svelte
<script>
  import { Activity, Star } from '@jis3r/icons';
</script>

<Activity size={32} color="orange" strokeWidth={2.5} />
<Star size={32} color="blue" />
```

<Tip>
All icons animate on hover by default. Pass the `animate` prop to control animation state programmatically for advanced use cases.
</Tip>
